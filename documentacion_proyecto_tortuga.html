1. Objetivo

Disenar una tortuga en 2D/3D cuyo equilibrio se produzca en el hocico y justificarlo con centro de masa.

3. Tecnologias usadas

  • Python 3 + NumPy para integracion numerica.
  • OpenSCAD para modelado 3D parametrico.
  • HTML/CSS para documentacion tecnica.

4. Modelo geometrico 2D

La region 2D de la tortuga se define como union de:

  1. Caparazon: elipse.
  2. Cabeza: circulo.
  3. Hocico: triangulo (apoyo en (0,0)).
  4. Cola: triangulo.
  5. Aletas delanteras y traseras: triangulos.
  6. Dos engrosamientos frontales: circulos de radio r.

5. Centro de masa 2D (integrales dobles)

M   = rho * double_integral_R( dA )
Mx  = rho * double_integral_R( x dA )
My  = rho * double_integral_R( y dA )
x_bar = Mx / M = double_integral_R( x dA ) / double_integral_R( dA )
y_bar = My / M = double_integral_R( y dA ) / double_integral_R( dA )

Se calibra r hasta cumplir x_bar ~= 0 y y_bar ~= 0 en el hocico.

6. Volumen y centro de masa 3D (integrales triples)

El modelo 3D realista no es una extrusion plana: se construye como combinacion de elipsoides y capsulas (union/diferencia booleana). Por eso se evalua con integracion numerica tipo Monte Carlo: 

V ~= V_box * (N_inside / N_total)
x_bar ~= promedio( x_i ) sobre puntos dentro del solido
y_bar ~= promedio( y_i ) sobre puntos dentro del solido
z_bar ~= promedio( z_i ) sobre puntos dentro del solido

El script calculo_centro_masa_3d_tortuga.py implementa esta verificacion y calibra la escala frontal para que x_bar ~= 0 en el hocico.

7. Como ejecutar

7.1 Recalcular y generar vista 2D

python3 calculo_centro_masa.py --solve --plot vista_2d_tortuga.svg

7.2 Abrir modelo 3D en OpenSCAD

open -a /Applications/OpenSCAD-2021.01.app modelo_tortuga.scad

Si la app tiene otro nombre: open -a /Applications/OpenSCAD.app modelo_tortuga.scad.

7.3 Verificar centro de masa 3D (modelo organico)

python3 calculo_centro_masa_3d_tortuga.py --samples 2000000
  1. Preview rapido: tecla F5.
  2. Render final: tecla F6.
  3. Exportar STL: File -> Export -> Export as STL.
