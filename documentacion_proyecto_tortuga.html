Calculo 2 + Modelado 3D

Tortuga 3D con centro de masa alineado al apoyo universal

Documento de desarrollo matematico: ecuaciones del objeto, integrales dobles y triples, criterio de estabilidad y visualizacion 3D embebida.

La idea central del proyecto es esta: mover masa hacia la parte delantera para que la proyeccion del centro de masa coincida con el eje de apoyo. Por eso las aletas delanteras y los lobulos frontales son grandes: aumentan el momento positivo en x y desplazan el centro de masa hacia la cabeza.

1. Sistema de coordenadas y objetivo

Se toma el origen en el eje comun de apoyo. En la vista en planta, el eje x apunta hacia la cabeza, el eje y hacia los costados y el eje z hacia arriba. La condicion buscada es:

x̄ ≈ 0, ȳ ≈ 0 para que la proyeccion del centro de masa caiga sobre el eje de apoyo.

Para apoyo plano adicionalmente se exige que la proyeccion del centro de masa quede dentro del area de apoyo. Para apoyo puntual en lapiz se exige que el centro de masa quede por debajo del centro de curvatura de la cavidad esferica.

2. Modelo analitico 2D para el desarrollo a lapiz y papel

El modelo 2D se define como una union de regiones simples. Esta es la parte que mejor se presta para el desarrollo manual de Cálculo 2.

E = { (x,y) : ((x + 3.4) / 3.1)² + (y / 2.2)² ≤ 1 } H = { (x,y) : (x + 1.0)² + y² ≤ 0.95² } C⁺(r) = { (x,y) : (x - 4.8)² + (y - 2.1)² ≤ r² } C⁻(r) = { (x,y) : (x - 4.8)² + (y + 2.1)² ≤ r² } R(r) = E ∪ H ∪ B ∪ T ∪ F⁺ ∪ F⁻ ∪ R⁺ ∪ R⁻ ∪ C⁺(r) ∪ C⁻(r)

Los triangulos del hocico, cola y aletas se toman con los vertices definidos en el archivo calculo_centro_masa.py. Por simetria respecto al eje x, se tiene inmediatamente:

Mᵧ(r) = ∬R(r) y dA = 0 ⇒ ȳ(r) = 0

3. Integrales dobles y centro de masa 2D

A(r) = ∬R(r) dA Mx(r) = ∬R(r) x dA My(r) = ∬R(r) y dA x̄(r) = Mx(r) / A(r) ȳ(r) = My(r) / A(r)

Para el desarrollo manual, cada pieza simple aporta area y primer momento. Si la pieza tiene area Aᵢ y centroide (xᵢ, yᵢ), entonces:

Mx = Σ Aᵢ xᵢ, My = Σ Aᵢ yᵢ

Los dos circulos delanteros son la variable de diseño. Como sus centros estan en x = 4.8, su aporte a Mx es positivo y por eso “jalan” el centro de masa hacia la cabeza.

4. Formulas exactas de las piezas simples

Pieza Area Centroide en x
Elipse del caparazon A = πab = π(3.1)(2.2) x = -3.4
Circulo de la cabeza A = πr² = π(0.95)² x = -1.0
Triangulo A = |det(P₂-P₁, P₃-P₁)| / 2 x = (x₁ + x₂ + x₃) / 3
Cada circulo delantero A = πr² x = 4.8
A(r) = A₀ + 2πr² Mx(r) = Mx,0 + 2πr²(4.8)

En el modelo analitico simplificado sin corregir traslapes, la solucion manual da:

r = √( - Mx,0 / (9.6π) ) ≈ 1.690416

Luego se refina sobre la union geometrica exacta con la grilla del script y se obtiene el valor final:

r ≈ 1.623037, x̄ ≈ 0.000031, ȳ ≈ 0.000007

5. Del plano al solido 3D

Para la pieza real ya no basta una extrusion plana. Se modela un solido organico por union y diferencia de elipsoides, capsulas y superficies de apoyo. La definicion exacta usada por el script es:

E₁ = { (x,y,z) : ((x + 2.0)/1.9)² + (y/1.95)² + ((z - 0.25)/1.25)² ≤ 1 } E₂ = { (x,y,z) : ((x + 2.1)/1.52)² + (y/1.55)² + ((z - 0.95)/1.0)² ≤ 1 } Capsula(P₀, P₁, r) = { Q : dist(Q, segmento(P₀,P₁)) ≤ r } Anillo = { (x,y,z) : 0.24² ≤ x² + y² ≤ 0.55², |z + 0.22| ≤ 0.08 } Cavidad = { (x,y,z) : x² + y² + (z - 0.58)² ≤ 0.94² }
T = (Caparazon ∪ Cabeza ∪ Cuello ∪ Aletas ∪ Cola ∪ LobulosFrontales ∪ Anillo) \ Cavidad

Esa definicion es rigurosa y es la que usa el verificador numérico. El archivo modelo_tortuga_estable.scad implementa la misma idea en OpenSCAD.

6. Volumenes de las piezas que mueven el equilibrio

La pieza clave son los dos lobulos delanteros. Cada uno es un elipsoide de semiejes 1.2s, 0.82s y 0.36s.

Vlobulo(s) = (4/3)π(1.2s)(0.82s)(0.36s) Vlobulo(1.664025) ≈ 6.852459

Como ambos lobulos estan centrados en x = 3.55, aportan un momento positivo grande:

Mx,lobulos(s) = 2 · Vlobulo(s) · 3.55

Esa es la razon matematica de que las “manos delanteras” y los engrosamientos frontales sean grandes: su funcion no es solo estetica, sino desplazar masa hacia adelante para que el centro de masa coincida con el eje de apoyo.

7. Integrales triples del centro de masa

χT(x,y,z) = 1 si (x,y,z) pertenece al solido T χT(x,y,z) = 0 en otro caso V = ∭ χT(x,y,z) dV Mx = ∭ x χT(x,y,z) dV My = ∭ y χT(x,y,z) dV Mz = ∭ z χT(x,y,z) dV x̄ = Mx / V, ȳ = My / V, z̄ = Mz / V

Como la geometria final incluye traslapes, uniones y una diferencia booleana por la cavidad, la evaluacion se hace con integracion de Monte Carlo sobre la funcion caracteristica. Eso sigue siendo un uso legitimo de integrales triples: solo cambia el metodo de evaluacion.

8. Criterios de estabilidad

Para que la pieza no solo tenga torque cero sino equilibrio util, se impusieron dos criterios adicionales.

Mesa / dedo ancho: √(x̄² + ȳ²) < Ranillo = 0.55
Lapiz / apoyo puntual: z̄ < zcentro de curvatura = 0.58

Con la configuracion universal actual, los resultados son:

V ≈ 35.793248 x̄ ≈ -0.000496 ȳ ≈ 0.004452 z̄ ≈ 0.143180 √(x̄² + ȳ²) ≈ 0.004480
A escala final de impresion (11 mm por unidad CAD), el desfase horizontal del centro de masa es de apenas 0.049 mm. La pieza queda estable sobre mesa, sobre dedo ancho y tambien sobre punta de lapiz gracias a la cavidad esferica.

14. Lectura corta para la exposicion

Primero se construyo un modelo 2D de la tortuga como union de una elipse, un circulo, varios triangulos y dos circulos delanteros de radio variable. Con integrales dobles se calcularon area y primer momento, y se ajusto el radio hasta forzar que el centro de masa quedara sobre el eje de apoyo. Luego se paso a un modelo 3D organico definido por elipsoides, capsulas, un anillo plano y una cavidad esferica. Sobre ese solido se aplicaron integrales triples para volumen y centro de masa mediante una funcion caracteristica e integracion numerica. El resultado final es una tortuga cuyo centro de masa queda alineado con el apoyo y que ademas se estabiliza sobre mesa, dedo y punta de lapiz.
